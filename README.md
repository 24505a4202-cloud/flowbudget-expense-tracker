# FlowBudget 💰

FlowBudget is a modern personal finance tracker that helps users monitor income, track expenses, visualize cash flow, and estimate future savings.

---

## 🚀 Features

* Add and manage income sources
* Track daily expenses
* Expense category classification
* Monthly income vs expense overview
* Net savings calculation
* Payment calendar
* Upcoming transaction timeline
* Built-in calculator
* Savings projection estimator
* Modern dashboard UI

---

## 📊 Dashboard Modules

### Overview

Shows:

* Total income
* Total expenses
* Net balance

### Add Transactions

Users can:

* Add income
* Add expenses
* Categorize transactions
* Add notes and dates

### Expenses List

Displays all transactions with:

* type (income/expense)
* date
* category
* frequency
* amount

### Calendar

Visual payment calendar showing:

* upcoming bills
* income dates
* expense events

### Savings Projection

Estimate future savings based on:

* monthly income
* monthly expenses
* time horizon
